# Boston Housing EDA

This project explores the Boston Housing dataset using Exploratory Data Analysis (EDA) using Python.

---

## 📊 Key Insights
- More rooms (RM) → Higher house prices  
- Higher LSTAT → Lower house prices  
- Strong relationships exist between features  

---

## 🛠 Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn
